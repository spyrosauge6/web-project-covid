# web-project-covid
Website wit﻿﻿h statistics of crowdsourced traffic data in order to avoid spreading of COVID-19
Αυτό το project αναπτύχθηκε στο πλαίσιο του μαθήματος Web Programming (CEID, Πανεπιστήμιο Πατρών) κατά τη διάρκεια του COVID-19.
Πρόκειται για μια web εφαρμογή καταγραφής και οπτικοποίησης κρουσμάτων COVID-19, βασισμένη σε PHP και MySQL, με δυνατότητα χρήσης χάρτη, 
καταχώρησης κρουσμάτων, διαχείρισης προφίλ, και οπτικοποίησης δεδομένων.

Σκοπός της εφαρμογής να επιτρέπει στους χρήστες: 
να βλέπουν σημεία με δηλωμένα κρούσματα σε χάρτη, 
να καταχωρούν δικές τους επισκέψεις, 
να ενημερώνονται αν βρέθηκαν σε επικίνδυνη τοποθεσία, 
να διαχειρίζονται το προφίλ τους, 
να βλέπουν στατιστικά και γραφήματα, 
να χρησιμοποιούν ένα πλήρες login/registration σύστημα

👤 Χρήστες
Εγγραφή, σύνδεση, έξοδος,
Προβολή επισκέψεων σε χάρτη,
Δήλωση θετικού κρούσματος,
Προειδοποίηση σε περίπτωση επαφής με θετικό κρούσμα,
Προβολή γραφημάτων & πινάκων

🔐 Admin
Εισαγωγή δεδομένων (Stores, COVID cases)
Διαχείριση χρηστών,
Προβολή στατιστικών,
Εισαγωγή επισκέψεων μέσω CSV
